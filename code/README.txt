Thư mục chứa logo nguồn cho icon ứng dụng.

1. Đặt file PNG vuông tại: app_icon.png
   - Khuyến nghị: 1024 × 1024 px (nền rõ, không trong suốt nếu đưa lên App Store).
2. Tạo lại icon Android / iOS:
   dart run flutter_launcher_icons

File hiện tại có thể là bản copy từ icon mặc định — thay bằng logo của bạn rồi chạy lệnh trên.
